## 🏁 Overview
A high-performance, immersive web application dedicated to Formula 1. This project goes beyond static pages by integrating interactive 3D WebGL models and high-end animations to deliver a premium, scrollytelling experience. It explores the aesthetics of modern F1 cars, including detailed features like Hamilton's Ferrari, while maintaining a smooth user interface.

## ✨ Features
* **Interactive 3D Models:** Integrated interactive F1 car models allowing users to explore the aerodynamic details up close.
* **Immersive Scrollytelling:** Utilizes parallax effects and smooth scroll animations to guide the user through the content dynamically.
* **Performance Optimized:** Built with responsiveness and SEO in mind. Heavy elements, such as native 3D text within the Spline models, were intentionally removed and replaced with standard, accessible web text to ensure fast load times and cross-device compatibility.

## 🛠️ Tech Stack
* **Framework:** React + TypeScript 
* **Build Tool:** Vite
* **3D & Animation:** Three.js / Spline, GSAP (GreenSock)
* **Styling:** CSS3 / Tailwind (Adjust if using a specific CSS framework)

## 🚀 Run Locally

**Prerequisites:** Ensure you have Node.js installed on your machine.

1. Clone the repository:
   ```bash
   git clone [https://github.com/Sharvi006/FORMULA-1.git](https://github.com/Sharvi006/FORMULA-1.git)

2. Install dependencies:
   `npm install`
3. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
4. Run the app:
   `npm run dev`
